# LogUtil-Android
Android library for working with Log file in Android SDK.


Add it in your root build.gradle at the end of repositories:

  	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}


Add the dependency

	dependencies {
	        implementation 'com.github.D4rkC00d3r:LogUtil-Android:Tag'
	}
  
Sync project.
